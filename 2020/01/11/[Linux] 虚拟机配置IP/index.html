[Linux] 虚拟机配置IP

发表于 2020-01-11

1、编辑网卡的配置文件
vim /etc/sysconfig/network-scripts/ifcfg-ens33
2、添加配置信息
IPADDR=10.10.28.192(要和本机IP地址同一个网段 即Windows系统doc窗口输入命令ipconfig第三个IP地址)

NETMASK=255.255.255.0
GATEWAY=10.10.28.1(非必要)
DNS1=8.8.8.8
3、修改配置信息
ONBOOT=yes
BOOTPROTO=static

注意：如果重新加载虚拟机快照，需要修改 HWaddr 的值（确保与虚拟机一致）
4、重启网卡
service network restart
